# Project Roadmap

The following list outlines planned features, interaction improvements, and known bugs for the Tasks Graph application.

## Planned Features & Interactions
- [ ] **Workflow Management**: Make workflow items selectable and draggable for reordering.
- [ ] **Task Completion**: Ability to mark tasks as completed and view them in a dedicated "Finished" area.
- [ ] **Visual Clarity**: Make nodes semi-transparent so dependency lines are visible underneath cards.
- [ ] **Global Search**: Search for specific nodes across all existing workflows.
- [ ] **Feedback**: Integrate haptic feedback for gestures and interactions.
- [ ] **Gestures**: 
  - Swipe right gesture to open the workflows menu.
  - Pinch-to-zoom support for the graph view.
- [ ] **Selection Mechanics**: Improved mechanics for easier selection of multiple nodes simultaneously.
- [ ] **Design Polish**: Refine hover border behaviour to prevent node resizing.

## Known Bugs & Fixes
- [ ] Fix node positioning jumps on iOS when renaming tasks.
- [ ] Resolve interface issues when moving nodes between workflows on iOS.
- [ ] Stabilize hover border sizing consistency across columns.
